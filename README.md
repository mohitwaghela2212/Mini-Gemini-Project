# Mini Gemini Project

A full-stack project built with React (Vite) for the frontend and FastAPI for the backend.

## Features
- Modern React frontend
- FastAPI backend
- Easy local development setup

## Project Structure
- `frontend/` - Vite + React app
- `backend/` - FastAPI application

## Setup Instructions

### Backend
1. Open a terminal in the `backend` folder.
2. Create and activate a virtual environment.
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the server:
   ```bash
   uvicorn main:app --reload
   ```

### Frontend
1. Open a terminal in the `frontend` folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Notes
- Make sure both frontend and backend are running for the full application experience.
- Update environment variables if your project uses them.
